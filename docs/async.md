–ó–µ–ª–µ–Ω—ã–π –ø–æ—Ç–æ–∫ ‚Äî —ç—Ç–æ –æ–±—ã—á–Ω—ã–π –ø–æ—Ç–æ–∫, –∑–∞ –∏—Å–∫–ª—é—á–µ–Ω–∏–µ–º —Ç–æ–≥–æ, —á—Ç–æ –ø–µ—Ä–µ–∫–ª—é—á–µ–Ω–∏—è –º–µ–∂–¥—É –ø–æ—Ç–æ–∫–∞–º–∏ –ø—Ä–æ–∏–∑–≤–æ–¥—è—Ç—Å—è –≤ –∫–æ–¥–µ –ø—Ä–∏–ª–æ–∂–µ–Ω–∏—è, –∞ –Ω–µ –≤ –ø—Ä–æ—Ü–µ—Å—Å–æ—Ä–µ.   
`Gevent` ‚Äî –∏–∑–≤–µ—Å—Ç–Ω–∞—è Python-–±–∏–±–ª–∏–æ—Ç–µ–∫–∞ –¥–ª—è –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è –∑–µ–ª–µ–Ω—ã—Ö –ø–æ—Ç–æ–∫–æ–≤ –Ω–µ–±–ª–æ–∫–∏—Ä—É—é—â–µ–≥–æ –≤–≤–æ–¥–∞-–≤—ã–≤–æ–¥–∞ `Eventlet`. `Gevent.monkey` –∏–∑–º–µ–Ω—è–µ—Ç –ø–æ–≤–µ–¥–µ–Ω–∏–µ —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω—ã—Ö –±–∏–±–ª–∏–æ—Ç–µ–∫ Python —Ç–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º, —á—Ç–æ –æ–Ω–∏ –ø–æ–∑–≤–æ–ª—è—é—Ç –≤—ã–ø–æ–ª–Ω—è—Ç—å –Ω–µ–±–ª–æ–∫–∏—Ä—É—é—â–∏–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ –≤–≤–æ–¥–∞-–≤—ã–≤–æ–¥–∞.   
–í–æ—Ç –ø—Ä–∏–º–µ—Ä –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è `Gevent` –¥–ª—è –æ–¥–Ω–æ–≤—Ä–µ–º–µ–Ω–Ω–æ–≥–æ –æ–±—Ä–∞—â–µ–Ω–∏—è –∫ –Ω–µ—Å–∫–æ–ª—å–∫–∏–º URL-–∞–¥—Ä–µ—Å–∞–º:
```
import gevent.monkey
from urllib.request import urlopen
gevent.monkey.patch_all()
urls = ['http://www.google.com', 'http://www.yandex.ru', 'http://www.python.org']

def print_head(url):
    print('Starting {}'.format(url))
    data = urlopen(url).read()
    print('{}: {} bytes: {}'.format(url, len(data), data))

jobs = [gevent.spawn(print_head, _url) for _url in urls]

gevent.wait(jobs)
```
–ö–∞–∫ –≤–∏–¥–∏—Ç–µ, API-–∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å Gevent –≤—ã–≥–ª—è–¥–∏—Ç —Ç–∞–∫ –∂–µ, –∫–∞–∫ –∏ –ø–æ—Ç–æ–∫–∏.   
–û–¥–Ω–∞–∫–æ –∑–∞ –∫–∞–¥—Ä–æ–º –æ–Ω –∏—Å–ø–æ–ª—å–∑—É–µ—Ç —Å–æ–ø—Ä–æ–≥—Ä–∞–º–º—ã (coroutines), –∞ –Ω–µ –ø–æ—Ç–æ–∫–∏, –∏ –∑–∞–ø—É—Å–∫–∞–µ—Ç –∏—Ö –≤ —Ü–∏–∫–ª–µ —Å–æ–±—ã—Ç–∏–π (event loop) –¥–ª—è –ø–æ—Å—Ç–∞–Ω–æ–≤–∫–∏ –≤ –æ—á–µ—Ä–µ–¥—å. –≠—Ç–æ –∑–Ω–∞—á–∏—Ç, —á—Ç–æ –≤—ã –ø–æ–ª—É—á–∞–µ—Ç–µ –ø—Ä–µ–∏–º—É—â–µ—Å—Ç–≤–∞ –ø–æ—Ç–æ–∫–æ–≤, –±–µ–∑ –ø–æ–Ω–∏–º–∞–Ω–∏—è —Å–æ–ø—Ä–æ–≥—Ä–∞–º–º, –Ω–æ –≤—ã –Ω–µ –∏–∑–±–∞–≤–ª—è–µ—Ç–µ—Å—å –æ—Ç –ø—Ä–æ–±–ª–µ–º, —Å–≤—è–∑–∞–Ω–Ω—ã—Ö —Å –ø–æ—Ç–æ–∫–∞–º–∏.
